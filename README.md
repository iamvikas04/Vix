# Vix-bot
This is a very simple discord bot

# Installation
* First we need to install Node in our system
* Then we need discord.js 

```cmd
npm install discord.js
```

# Setup
* Just add your discord tokan to config.json

```js
{
  "prefix": "!",
  "tokan": "you tokan here"
}
```

# Cammands

```cmd 
!hey
```
```cmd
!info
```

```cmd
!server
```
